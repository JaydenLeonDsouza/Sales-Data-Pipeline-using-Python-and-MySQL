# Sales-Data-Pipeline-using-Python-and-MySQL
Built a Python-based data pipeline that reads sales data from CSV files, performs revenue calculations and product-wise analysis using Pandas, visualizes results with Matplotlib, and stores processed data into a MySQL database using SQLAlchemy.
...Features
Reads sales data from CSV
Calculates revenue totals
Finds top earning product
Generates a revenue bar chart
Stores processed data into MySQL
...Tools Used
Python
Pandas
Matplotlib
SQLAlchemy
MySQL
